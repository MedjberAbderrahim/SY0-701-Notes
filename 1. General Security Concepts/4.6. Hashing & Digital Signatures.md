## Hashing
- Similar to encryption
- Hashing Functions Create a **unique, fixed size output** from input
### Uses of Hashing
- Verify authenticity of files
- Store salted hashes of passwords, instead of clear-text passwords
### Salt
- Random data added to a password when hashing
- Rainbow tables won't work with salted hashes
- Slows brute-force, doesn't stop reverse engineering
## Digital Signatures
- Proves Integrity, Authentication, and Non-Repudiation
- Sign -hash of file- using source's private key, verify using public key, if decrypted hash == hash(decrypted _content)
