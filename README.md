# CompTIA SY0-701 Security+ Notes

These are my notes for studying and passing CompTIA SY0-701 Security+, as I followed and studied along [Professor Messer](https://professormesser.com/)'s [Security+ youtube playlist](https://youtube.com/playlist?list=PLG49S3nxzAnl4QDVqK-hOnoqcSKEIDDuv&si=wZwg2meKrN1JpsUj).

The goal of publishing these notes is to provide learning material for people looking to pass the CompTIA SY0-701 Security+ or study its course material, or maybe some notes for them to revise or to compare against.

> [!note] Image & media display
> Most of these note include images defined in Obsidian format (`[[FileName]]`), to be able to see them correctly, please install Obsidian in your PC, a free markdown note taking app.
> If there exists any other platforms that support this format natively, or some fast, efficient, and automatable method to fix this formatting and make it more markdown-native (i.e. the `[FileName](FilePath)` format), I would be happy to know!


## Disclaimer

These notes do not guarantee one's success, and should be taken as mere revision notes, they have been optimized and revolved on my own pre-existing knowledge.

You can find some minor topics to be missing, as I deemed them unimportant, redundant, or "too obvious".

CompTIA's [SY0-701 Exam objectives](./comptia-security-sy0-701-exam-objectives-5-0-1.pdf) remains the sole official exam objectives document, and can be used to pick whichever domains you are lacking in.

All the assets have been taken from public & totally-free resources and websites, such as [Professor Messer](https://professormesser.com/)'s [Security+ youtube playlist](https://youtube.com/playlist?list=PLG49S3nxzAnl4QDVqK-hOnoqcSKEIDDuv&si=wZwg2meKrN1JpsUj), and public websites and/or documentation.
