### Types of obfuscation
#### Steganography
-  Hide/Embed information inside a media or smthng
-  "Security through Obscurity"
-  Can be Applied to:
	-  Images: Hiding data within an image
	-  Network Traffic: Embed data withing TCP Packets, partially send it
	-  Invisible watermarks: e.g. yellow dots on laser-printed paper, for printer identification
	-  Audio: Interlace a secret message within the audio
	-  Video: image steganography on a larger scale
#### Tokenization
-  Replace Sensitive Data with a non-Sensitive placeholder
-  Generate Temporary Token, that is one-time usable
-  Original data and token aren't mathematically related
#### Data Masking
-  Partially hide data, using substition (with '*') or shuffling
