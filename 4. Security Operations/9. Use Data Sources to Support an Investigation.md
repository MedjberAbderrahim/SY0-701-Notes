# Log Data
- Massive security-related information are stored in logs, e.g. exploit attempts, traffic flow, ...
- Can be collected from various sources:
	- Firewall: source/destination IP address, anomalies/suspicious data (NGFW), ...
	- Application: in linux it's generally stored in `/var/log`, generally parsed by SIEM
	- Endpoints: phones, laptops, logon events, ...
	- OS-Specific Security Events
	- IPS/IDS
	- Network Logs
	- Metadata
