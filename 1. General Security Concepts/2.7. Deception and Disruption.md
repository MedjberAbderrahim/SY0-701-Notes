## Honeypots
-  Attract the bad guys, and trap them there.
-  Can be used to lean into the techniques used by the attacks.
## Honeynet
Combine virtualized honeypots to create entire networks of fake devices, to make them look more realistic.
## Honeyfiles
Honeypots on the file level
-  Create files with fake information.
-  Bait for the **honeynet** (e.g. passwords.txt)
-  An alert can be sent **if the file is accessed**.
## Honeytokens
Pieces of data embedded within the network or systems that act as markers or traps. Helps identify data leaks or unauthorized access by tracking their usage or distribution.  
Such as:
- Fake API credentials placed on public cloud shares
- Fictitious email addresses that are monitored for appearance online
Honeytokens can be any data type, such as database records, browser cookies, or embedded pixels on web pages, which can be monitored if posted or accessed elsewhere.
